\item Los electrones tienden a llenar las subcapas de tal manera que las que se llenan primero son las subcapas con valores $n + \ell$ más pequeños. Si dos subcapas tienen el mismo valor de $n + \ell$, se llena primero aquella que posee el menor valor de $n$. Utilizando estas reglas, escriba el orden secuencial en que se llenan las subcapas hasta $n + \ell = 7$.
